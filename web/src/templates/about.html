Summary

My name is Stephen Kim and this is my ECE 140B website. I am currently a third-year Electrical Engineering student specializing in Machine Learning & Control. I recently got admitted into graduate school for Machine Learning & Data Science.

In this page, I will be discussing the features of this website as well as the accessibility features this website provides.

UI Principles

This website has the goal to be as minimalistic as possible and to display mainly content in an easy to read way.

Each aspect of this website is meant to be as consistent as possible upon every page. The sidebar exists on each page and is always in the same spot.

Each part of this website is consistent. The sidebar remains in the same spot and the routes are all in the same order.

The buttons within the sidebar all highlight when they are buttons and those that are not buttons will not highlight.

Since the goal of this website was to be very minimalistic and straight to the point, the primary goal highlights the clarity in what is presented on the webpages.

This website shows off a dark theme that is easy on the eyes while maintaining enough contrast to be distinguishable in greyscale.

This website is designed to perform at least decently on mobile, desktop, and tablet.

IA Principles

This website keeps the number of choices to a minimum especially when it comes to form submission that can change the database.

For objects on each webpage, similar groupings of objects are all consistent in styling, structure, and behavior.

There are a number of recurring objects on each webpage and each recurring object remains constient throughout the entire website. The state of the objects will always be predictable.

Each webpage has an IA element that will allow users to navigate everywhere on the website. No matter their starting point.

The single IA method to navigate the many webpages is through the sidebar.
